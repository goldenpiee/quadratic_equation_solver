# План Тестирования Quadratic Solver

## Описание
План тестирования для проверки функции `solve_quadratic_equation`, решающей квадратное уравнение.

## Тестовые случаи

# План тестирования

## Позитивные тесты

1. **Тест: Реальные корни при дискриминанте равном нулю**
    - Входные данные: a=1, b=-2, c=1
    - Ожидаемый результат: Корни (1, 1)

2. **Тест: Реальные корни при положительном дискриминанте**
    - Входные данные: a=1, b=-3, c=2
    - Ожидаемый результат: Корни (2, 1)

3. **Тест: Отсутствие реальных корней при отрицательном дискриминанте**
    - Входные данные: a=1, b=1, c=1
    - Ожидаемый результат: None

## Негативные тесты

4. **Тест: Неверный коэффициент a (a=0)**
    - Входные данные: a=0, b=1, c=1
    - Ожидаемый результат: Возникновение исключения ValueError

5. **Тест: Неверные коэффициенты a и b (a=b=0)**
    - Входные данные: a=0, b=0, c=1
    - Ожидаемый результат: Возникновение исключения ValueError

## Запуск тестов

1. Запустите тесты с использованием стандартной библиотеки `unittest`:
    ```bash
   python -m unittest discover -s C:\Users\qwe\Documents\GitHub\quadratic_equation_solver\tests -p 'test_*.py'

    ```
